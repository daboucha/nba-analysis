Our Findings

We began our project by scraping data from NBA.com. This resulted in an abundance of data. While we did not have to worry about missing data, we did have to investigate for players with the same name. Once the data was obtained, we stored the data into dictionaries and dataframes, dropping all columns that were not necessary for the prediction model. Ultimately, we created new dataframes to hold the average statistics of each individual player over the last 5 years.

  • Scikit-learn model used to train, test and split our data
  • Training Set: Last 10 years of each individual game statistics (dropping unecessary columns)

Lastly, we tried a few prediction models, such as Random Forest Regression and Multiple Linear Regression. Since we had labeled input and output data, we knew we would be using supervised learning. After testing the model and obtaining similar results, we determined that the most efficient for our prediciton was the Multiple Linear Regression Model.

  • Testing Set: Live data collected by web scraping
  • Feature Selection: we dropped all categorical data, and statistics that revealed the data we were trying to predict (i.e. Field Goals Made) and then dropped redundant or duplicated columns
